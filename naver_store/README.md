# 네이버 스마트 스토어 상세 설명란용 롱 이미지

네이버 스마트 스토어 상세 설명란에 사용하기 위한 단순화된 롱 이미지 형태의 랜딩페이지입니다.

## 구조

- `index.html`: 네이버 스마트 스토어 상세 설명란에 삽입할 HTML 파일

## 사용 방법

1. `index.html` 파일을 브라우저에서 열어 확인
2. 필요에 따라 내용 수정
3. 네이버 스마트 스토어 상세 설명란에 HTML 코드 삽입

## 주요 섹션

1. **히어로 섹션**: 메인 타이틀 및 소개
2. **문제점 섹션**: 사용자가 겪는 문제점 소개
3. **솔루션 소개**: GlucoUS의 주요 기능 설명
4. **베타 테스터 혜택**: 프로 플랜 기능 및 혜택 안내
5. **CTA 섹션**: 행동 유도 메시지

## 스타일

- Tailwind CSS 사용
- 반응형 디자인
- GlucoUS 브랜드 컬러 적용 (Primary: #18D9C2)
